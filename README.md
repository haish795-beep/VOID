# VOID BOT 공식 사이트

이 저장소의 `bot-site` 폴더는 VOID BOT 공식 정보 페이지입니다.

로컬 확인

- VS Code에서 `Live Server` 확장으로 열거나 간단한 정적 서버를 사용하세요.

예: Python 로컬 서버

```bash
# 파이썬 3
python -m http.server 8000
# 브라우저에서 http://localhost:8000/ 를 열고 bot-site 폴더로 이동
```

GitHub Pages 배포

1. GitHub 저장소를 생성하고 파일을 푸시합니다.
2. GitHub에서 `Settings` → `Pages`로 이동해 브랜치(예: `main` 또는 `gh-pages`)와 경로(`/ (root)` 또는 `/docs`)를 설정하세요.
3. 도메인 연결이 필요하면 `Custom domain`을 설정하세요.

수정 사항

- 사이트 텍스트, 링크(오픈채팅/오픈프로필), 연락처 이메일은 실제 값으로 교체하세요.
- `assets/logo.svg`를 원하는 로고로 교체하세요.

문의
